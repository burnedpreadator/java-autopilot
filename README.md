# java-autopilot

Java Autonomous Testing Platform

An open-source AI-powered testing platform for Java desktop applications.

## Vision

JATP aims to become the Playwright of Java desktop applications.

The platform can:

- Understand Java source code
- Analyze Eclipse RCP/SWT applications
- Read test case documents
- Launch applications automatically
- Navigate screens autonomously
- Execute test cases
- Validate runtime data
- Generate APIs for external execution
- Integrate with Robot Framework and CI/CD systems

## Supported Technologies

- Java 1.7+
- Eclipse RCP
- SWT
- Swing
- JavaFX
- OSGi
